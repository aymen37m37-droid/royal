✅ تم بنجاح تحويل المشروع من Frontend Only إلى نظام متكامل!

───────────────────────────────────────────────────────────
🚀 البدء السريع (5 دقائق)
───────────────────────────────────────────────────────────

1️⃣ إنشاء قاعدة البيانات:
   mysql -u root -p
   CREATE DATABASE restaurant_db;
   EXIT;

2️⃣ إعداد Laravel:
   cd backend
   composer install
   cp .env.example .env
   # عدل .env وأضف بيانات قاعدة البيانات
   php artisan key:generate
   php artisan migrate

3️⃣ تشغيل النظام:
   php artisan serve
   # ثم افتح index.html في المتصفح
